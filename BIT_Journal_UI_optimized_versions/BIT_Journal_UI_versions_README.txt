2025 中科院期刊分区查询 UI 优化版说明

1. BIT_Journal_UI_01_Official_Academic.html
   UI优化器推荐版，适合导师汇报、学院内部分享、正式科研查询。

2. BIT_Journal_UI_02_AI_CV_Tech_Dark.html
   AI/CV 科技暗色风，适合计算机、人工智能、视觉算法、工科实验室。

3. BIT_Journal_UI_03_Minimal_Literature_Search.html
   极简文献检索风，适合研究生日常查刊、快速投稿筛选、论文调研。

4. BIT_Journal_UI_04_Data_Admin_Light.html
   清爽数据管理风，适合科研秘书、课题组管理、跨学科老师共享查询。
